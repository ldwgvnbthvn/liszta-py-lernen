# DATA TYPES IN PYTHON
| Data types | Examples |
| --- | --- |
| Text type | ```str``` |
| Numeric type | ```int```, ```float```, ```complex``` |
| Sequence type | ```list```, ```tuple```, ```range``` |
| Mapping type | ```dict``` |
| Set type | ```set```, ```frozenset``` |
| Boolean type | ```bool``` |
| Binary type | ```bytes```, ```byearray```, ```memoryview``` |
| None type | ```NoneType``` |
